Die Asiatische Hornisse : Ein großer Feind der Bienen


Sie ist wie ihr Name schon sagt in Asien heimisch. Aber in einer speziellen Region dort. In Südostasien leben die Hornissen seit langem. Aber auch in Deutschland haben sie seit 2014 ein Zuhause gefunden. Die asiatische Hornisse ist etwas größer als die hier heimische Hornisse. Sie frisst für ihr Leben gern Bienen. Das macht sie so gefährlich für die Bienen.In der Fachsprache wird die asiatische Hornisse Vespa velutina nigrithorax genannt.
